iSports

iSports is a college sports website for students to read sports news, discuss ball games, make competition appointment and rent courts on campus, aiming to improve the students' daily sporting life. The website is based on PHP Laravel framework and bootstrap.

I was in charge of implement making game appointment function and renting courts function. Also, I helped my partners to optimize the website UI and realize the responsive web design.
